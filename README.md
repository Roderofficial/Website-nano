# NANO UI - Website for discord bot "NANO"

Strona powstała na w roku 2021. Jej celem było ułatwienie zarządzania botem discord *Nano*. 


## Features

- Panel logowania
- Historia wiadomości z discorda (te usunięte też)
- Podstawowe ustawienia bota
